数组

  • 数组可以存放多个同一类型的数据，实现对数据的统一管理。数组也是一种数据类型，是引用类型。数组型数据是 对象（Object）。

  • 数组中的元素可以是任何数据类型，包括基本类型与引用类型，但是不能混用。

  • 数组创建后，如果没有赋值，有默认值：int(0)，short(0)，byte(0)，long(0L)，float(0.0F)，double(0.0)，char(\u0000)，boolean(false)，String(null)，Object(null)。

  • 通过下标来访问数组的元素，下标从0开始。且必须在有效下标范围内使用，最小0，最大长度减一。

  • double nums[], 表示是double类型的数组， 数组名nums。

数组赋值机制

  • 基本数据类型赋值，赋值方式是值拷贝，这个值就是具体的数据。
    int n1 = 2; int n2 = n1;，n2的值不会影响n1的值

  • 数组在默认情况下是引用传递，赋的值是地址。（引用拷贝）

    因为数组在JVM的栈中指向一个地址，地址对应堆里的一个数据空间

数组添加/扩容

  • 当数组达到上限时，创建一个容量更大的新数组。将旧数组的元素依次放入，之后替换旧数组。

排序

  • 排序是将多个数据，依指定的顺序进行排列的过程。

内部排序

  • 指将需要处理的所有数据都加载到内部存储器中进行排序。
  • 包括(交换式排序法、选择式排序法和插入式排序法)；

外部排序

  • 数据量过大，无法全部加载到内存中，需要借助外部存储进行排序。
  • 包括(合并排序法和直接合并排序法)。

冒泡排序

  • 基本思想是：对待排序序列从后向前（从下标较大的元素开始）遍历，依次比较相邻元素的值，若发现前数大于后数则交换，使值较大的元素逐渐从前移向后部。
  • 每一轮排序可以确定一个数的位置，一共进行n-1次

二维数组

  • 定义形式：int[ ][ ]
  • 二维数组实际是由多个一维数组组成的，它的各个元素的长度可以相同，也可以不同，称为列数不等
    的二维数组。
  • 数组是一个对象，所以二维数组的元素存放的是一维数组的地址。
