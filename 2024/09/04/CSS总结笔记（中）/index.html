常用CSS属性

字体属性

属性 意义 注意点
font 简写属性。把所有针对字体的属性设置在一个声明中 按顺序用空格隔开, 必须保留 font-size 和 font-family 属性
font-size 字号 必须使用单位:px像素,默认16px; em倍数，相对于父标签字体大小的倍数
font-family 字体 指定多种字体时，用逗号分隔; 字体由多个单词组成时由双引号括起来
font-weight 粗细 Normal 为正常; lighter 为细体; bold 为粗体; bolder 为特粗体
font-style 样式(是否倾斜) Normal 为正常; italic 为斜体; oblique 为倾斜

文本属性

属性 意义 注意点
color 文本颜色 有四种写法
line-height 行高
text-align 水平对齐方式 取值：left、center、right(属性要设置给内容的父级)
vertical-align 垂直对齐方式 取值：top、middle、bottom(属性要设置给内容的父级)
text-indent 首行缩进 2em（推荐：1em = 当前标签的字号大小）
text-decoration 文本修饰 取值：underline、overline、line-through
text-transform 字母大小写转换 取值：lowercase、uppercase、capitalize首字母大写
letter-spacing 字符间距

行内元素的垂直外边距

背景属性

属性 意义 注意点
background-image 背景图 属性值：url(背景图 URL),默认有平铺（复制）效果。
background-repeat 平铺方式 repeat(默认)，repeat-x(水平平铺)，repeat-y, no-repeat
background-color 背景颜色 取值：transparent 透明
background-position 背景图位置 关键字：top、bottom、left、right、center; 坐标：左上角为(0,0)坐标，向右为x正方向,向下为y正方向
background-size 背景图缩放 cover：等比例缩放背景图片以完全覆盖背景区，可能背景图片部分看不见; contain：等比例缩放背景图片以完全装入背景区，可能背景区部分空白
background-attachment 背景图固定 作用：背景不会随着元素的内容滚动。属性值：fixed
background 背景复合属性 属性值：背景色 背景图 背景图平铺方式 背景图位置/背景图缩放 背景图固定（空格隔开各个属性值，不区分顺序）

列表属性

属性 意义
list-style 简写属性，用于设置列表项的标志、位置和图像
list-style-type 设置列表项的标志类型
list-style-position 设置列表项标志的位置
list-style-image 设置列表项的图像作为标志

表格属性

属性 描述
border-collapse 设置是否合并表格边框模型
border-spacing 设置相邻单元格边框之间的距离
caption-side 设置表格标题的位置
empty-cells 设置是否显示表格中的空单元格的边框和背景
table-layout 设置表格单元格、行和列的算法
border 简写属性，用于设置表格边框的宽度、样式和颜色
border-width 设置表格边框的宽度
border-style 设置表格边框的样式
border-color 设置表格边框的颜色
border-radius 设置表格边框圆角

盒子模型

网页布局过程：

  1. 先准备好相关的网页元素，网页元素基本都是盒子 Box 。
  2. 利用 CSS 设置好盒子样式，然后摆放到相应位置。
  3. 往盒子里面装内容。

属性

属性 描述 说明
width 设置盒子内容的宽度
height 设置盒子内容的高度
padding或padding-方位名词 设置盒子内边距 内容与边框之间的距离,添加 padding 会撑大盒子,多值写法:从上开始顺时针赋值，当前方向没有数值则与对面取值相同。
border或border-方位名词 设置盒子的边框 属性值：边框线粗细 线条样式(solid,dashed,dotted) 颜色（不区分顺序）
margin 设置盒子外边距 盒子与盒子之间的距离,与 padding 属性值写法、含义相同

默认情况：盒子尺寸 = 内容尺寸 + border 尺寸 + 内边距尺寸

结论：给盒子加 border / padding 会撑大盒子

解决：

  • 手动做减法，减掉 border / padding 的尺寸
  • 內减模式：box-sizing: border-box

版心居中

圆角

作用：设置元素的外边框为圆角。

属性名：border-radius

属性值：数字+px / 百分比

提示：属性值是圆角半径,多值写法同上

阴影

作用：给元素设置阴影效果

属性名：box-shadow

属性值：X 轴偏移量 Y 轴偏移量 模糊半径 扩散半径 颜色 内外阴影

注意：

  • X 轴偏移量 和 Y 轴偏移量 必须书写
  • 默认是外阴影，内阴影需要添加 inset

清除默认样式

网页元素很多都带有默认的内外边距，而且不同浏览器默认的也不一致。因此我们在布局前，首先要清除下网页元素的内外边距。

元素溢出

作用：控制溢出元素的内容的显示方式。

属性名：overflow

属性值: hidden(溢出隐藏),scroll(溢出滚动,始终显示滚动条),auto(溢出滚动,溢出才显示滚动条)

外边距问题

外边距合并现象

场景：垂直排列的兄弟元素，上下 margin 会合并

现象：取两个 margin 中的较大值生效

外边距塌陷问题

场景：父子级的标签，子级的添加 上外边距 会产生塌陷问题

现象：导致父级一起向下移动

解决方法：

  • 取消子级margin，父级设置padding
  • 父级设置 overflow: hidden
  • 父级设置 border-top
