Java学习笔记·程序控制结构

顺序控制

  • 程序从上到下逐行执行，中间没有任何判断和跳转（默认的控制顺序）
  • Java 定义变量时采用合法的前向引用 (先定义再使用)

分支控制

单分支

  • 基本语法，如果{}内只有一条语句，可省{}（但建议写上）
  • 条件表达式为Ture时，执行{}内代码，否则反之

双分支

  • 当条件表达式为 ture，就会执行 {执行代码块1;}；如果为 false 则执行 {执行代码块2;}。

多分支

  • 只有上一个条件不成立时才会判断下一个条件。
  • 多分支可以没有 else。此时如果条件都不成立，则无执行入口。
  • 若有else,若所有条件表达式都不成立，则默认执行else中代码块。

嵌套分支

  • 在一个分支结构中又完整的嵌套了另一个完整的分支结构

  • 里面的分支的结构称为内层分支，外面的分支结构称为外层分支

  • 嵌套不要超过三层

Switch分支结构

  1. switch 关键字，表示 switch 分支。
  2. 表达式 对应一个值，有返回值的就是表达式。该值必须是 int 或 char（char 可以转化为 int）。是 byte 或 short 的场合，要提升为 int。不允许 long、double 或 float
  3. case 常量1; 表示：当 表达式 的值等于 常量1 ，则执行 语句块1。
  4. break; 表示退出 switch 分支,并非退出程序
  5. 当 表达式 的值匹配 常量1 ，则执行 语句块1，如果不匹配，则继续匹配 常量2 ，以此类推。
  6. 如果全都不匹配，则执行 default。
  7. 如果不写 break; ，则会发生穿透，即不进行判断而继续执行下一语句块。

细节

  1. 表达式 数据类型，应和 case 后的 常量 类型一致，或者该常量可以自动转换成可以比较的类型。如：输入的是 char 而 常量 是 int

  2. switch （ 表达式 ）中表达式的返回值必须是：byte short int char ``enum String

  3. case 语句中的值必须是 常量 或 常量表达式，不能是 变量。

  4. default 是可选的。没有就不执行。

  5. break; 用来跳出 switch 分支。如果不写，会持续执行语句，直到分支结束或遇到下一个 break;

与if-else的取舍

  1. 如果判断的具体数值不多，而且符合byte、short 、int、char, enum[枚举], String 这6 种类型。虽然两个语句都可以使用，建议使用swtich 语句。
  2. 其他情况：对区间判断，对结果为boolean 类型判断，使用if，if 的使用范围更广

循环控制

for循环控制

  • 基本语法

  • 细节

    1. 循环条件是返回一个布尔值的表达式

    2. for(;循环判断条件;) 中的初始化和变量迭代可以写到其它地方，但是两边的分号不能省略。

    3. 循环初始值可以有多条初始化语句，但要求类型一样，并且中间用逗号隔开，循环变量迭代也可以有多条变量迭代语句，中间用逗号隔开。

while循环控制

  1. 循环条件是返回一个布尔值的表达式,不能为空
  2. while 循环是先判断再执行语句

dowhile循环控制

  • 循环条件是返回一个布尔值的表达式
  • 先执行再判断**，也即至少执行一次

多重循环控制

  • 将一个循环放在另一个循环体内，就形成了嵌套循环。其中，for ,while ,do…while 均可以作为外层循环和内层循环。（一般使用两层，最多不要超过3 层）

  • 嵌套循环 是把 内层循环 当成 外层循环 的 循环体。只有内层 false 时才可能结束当层循环。

  • 若内层执行 n 次，外层 m 次，则合计会循环 n*m 次

跳转控制语句

break

  • 用于中止一个语句块的执行

  • break 可以被用在三种场合中

    • switch 语句中，以跳出判断（结束穿透）

    • for、while、do…while 循环语句中，以跳出循环

    • 语句块中，以跳过本块中所有剩余语句

      break 语句出现在多层嵌套的语句块中时，可以通过 标签(命名随意) 指明要终止的时哪一层语句块。(尽量不要使用标签)没有写标签时默认退出最近的循环体

continue

  • 在循环中出现。用于结束本次循环，跳过剩下的语句，继续下一次循环
  • 进行下次循环前，仍会判断循环条件是否满足
  • 在多层嵌套循环中，可以通过标签指出跳出哪次循环（同 break）

return

  • 用于方法。表示跳出所在的方法
  • 方法有返回值的场合，将返回值写在 return 后：return 值;
  • 如果写在 主方法 则跳出程序
