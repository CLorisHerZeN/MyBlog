变量简介

  • 变量：变量是程序的基本组成单位
  • 变量的三个基本要素：类型 + 名称 + 值
  • 示例：int a = 1; 或int a;a = 1;类型 int 名称 a 值 1
  • 基本步骤：声明变量（告诉编译器变量的名称和数据类型，但不分配内存。） -> 赋值

原理与注意事项

  • 变量在内存中占据一定的空间，这个空间用于存储变量的值。内存分配通常发生在变量声明的时候。
  • 变量名实际上是内存地址的别名，这使得我们可以通过变量名来访问和操作存储在特定内存地址的数据。（变量名——>储存值的空间）
  • 变量在同一作用域内不能重名。
  • 每个变量都有一个数据类型，它决定了变量可以存储哪种类型的值，以及该值在内存中所占用的空间大小。

+号的使用

  1. 当左右两边都是数值型，做加法运算
  2. 当左右两边任意一方为字符串，做拼接运算
  3. 运算从左到右

数据类型

  • [8]表示占用内存空间（字节）

整数类型

  • 即使储存的值相等，数据类型不一样，空间就不一样
  • 占用存储空间不同，可以存储的数据的范围不同
类型 占用存储空间 范围
byte [字节] 1字节 -128 ~ 127
二进制
short [短整型] 2字节 -(2^15) ~ 2^15-1
-32768 ~ 32767
int [整型] 4字节 -2^31 ~ 2^31-1
-2147483648 ~ 2147483647
long [长整型] 8字节 -2^63 ~ 2^63-1

使用细节

  1. Java 各整数类型有固定的范围和字符长度，不受具体 OS（操作系统）影响，以保证 Java 程序的可移植性。

  2. Java 默认整型常量为 int ，要声明 long 型常量必须后加 l 或 L。除非不足以表示大数字，才使用long。

  3. bit: 计算机中的最小存储单位。byte(字节)：计算机中的基本存储单元，1byte = 8 bit。

浮点类型

类型 占用存储空间 范围
单精度float 4字节 -3.403E38 ~ 3.403E38
双精度double 8字节 -1.798E308 ~ 1.798E308

浮点数在机器中存放形式为：浮点数 = 符号位 + 指数位 + 尾数位

因此，尾数部分可能丢失，造成精度损失。换言之，小数都是近似值

使用细节

  1. 与整数类型相似，有固定的范围和字符长度，不受具体 OS（操作系统）影响。

  2. Java 默认浮点常量为 double ，要声明 float 型常量必须后加 ”f“ 或 ”F“

  3. 浮点型常量有两种表示形式

    十进制数形式：5.13、315.4F（表示为单精度的）、.414（可省略个位数的零）

    科学计数法：5.12e2 即[5.12 × 10^2]、5.12E-2 即[5.12 / 10^2]

  4. 通常情况下，应该使用 double 类型，以其更为精确。

  5. 浮点数使用陷阱：当我们对运算结果是小数的进行相对判断时，要小心。（因为*小数都是近似值*）

    正确方法是：以两个数差值的绝对值，在某个精度范围内判断

字符类型(char)

  • 可以存放单个字符。（可以存放一个数字，因为其字符是数字编号的。输出时会输出数字对应的字符。”编码的概念“）
  • char c1 = 'a'; char c2 = '\t'; char c3 = '字'; char c4 = 99;

使用细节

  1. 字符常量，也被称为字符字面量，是char 类型的具体取值。

    用来直接表示单个字符的值。用单引号括起来，若用双引号则表示字符串。

  • 以下是几种常见的字符常量类型：

    标准字符常量：这种字符常量由一个字符组成。例如，’A’ 是一个标准字符常量。

    转义字符常量：以反斜杠（\）开头的字符序列，用于表示非打印字符或其他具有特定含义的字符。例如，’\n’ 通常代表换行符，而 ‘\t’ 代表制表符。（两个合起来表示一个字符）

    多字符常量：虽然大多数编程语言只支持单字符常量，但有些语言允许使用多个字符来表示特定的值。这些字符通常具有特殊的含义，如 C 语言中的字符串转换指定符 ‘ll’ 用于 long long int。

    Unicode 字符常量：在支持 Unicode 的编程语言中，可以使用十六进制或 Unicode 编码来表示字符常量。例如，U+’0041’ 表示拉丁大写字母 A。

布尔类型（Boolean）

  • 只允许取值true或false，无null
  • 占一个字节
  • 适用于逻辑运算
    • if条件控制语句
    • while循环控制语句
    • do-while循环控制语句
    • for循环控制语句

编码

  • 字符类型的本质，是把字符对应的码值编程二进制，存储。显示时将二进制代码转化为码值，找到对应的字符。

  • 字符与码值的对应关系是字符编码表规定的。

    ASCII 编码表，占用 1 byte，共有 128 个字符。(实际一个字节可以表示256个字符)

    Unicode 编码表，占用 2 byte，字母汉字都占用 2 byte，这样可能浪费空间。0 - 127 的字符与 ASCII 相同，所以兼容 ASCII。

    UTF-8 编码表，根据不同符号大小可变（1 - 6 byte），字母占用 1 byte，汉字占用 3 byte。是 Unicode 的改进，是互联网上使用最广的 Unicode 实现方式。

    GBK 编码表，可以表示汉字，字母占用 1 byte，汉字占用 2 byte。

    GB2312 编码表，可以表示汉字（GB2312 < GBK）

    BIG5 编码表，可以存放繁体中文（香港，台湾）

基本数据类型转换

自动类型转换

  • 自动类型转换：Java 在进行赋值或运算时，精度（容量）小的类型自动转换为精度（容量）大的类型。

    char– > int –> long –> float –> double

    byte –> short –> int –> long –> float –> double

    例子：int a = 'c' 或者 double b = 80

自动类型转换细节

  1. 有多种类型数据混合运算时，系统会将所有数据转换成容量最大的那种，再进行运算。

  2. 如若把大精度（容量）数据赋值给小精度（容量）类型，就会报错（小数由于精度原因，大赋小会丢失精度，必不可用。int n2 = 1.1;//错误double -> int

    但整数大赋小时：

    1.赋予具体数值时，判断范围。

    2.变量赋值时，判断类型。反之进行自动类型转换。

  3. （byte short ）和char 之间不会自动转换，但可以计算。

    三者参与计算时首先转化为 int。

    byte a = 1;

    byte b = 1;

    a + b 结果是 int 类型

  4. boolean 类型不参与自动转换

  5. 自动提升原则：表达式结果的类型自动提升为(取决于)操作数中最大的类型。

强制类型转换

  • 自动类型转换的逆过程，将容量大的转换为容量小的
  • 使用时加强制转换符
  • 注意后果是精度降低或溢出

强制类型转换细节

  1. 强转符号只对最近的操作数有效，使用小括号提升优先级(先把括号内计算完毕，再应用强转，否则只对3有效)

  2. char 可以保留 int 的常量值，但不能保存其变量值。此时需要强制类型转换。

  3. byte short char 在进行运算时，当作 int 处理。

String（字符串）和基本类型转换

  • 基本数据类型转字符串

  • 字符串转基本数据类型（体验）,通过基本数据类型的包装类调用 parseXX 方法。

注意事项

  1. 将 String 转成基本数据类型时，要保证其能转换为有效数据。即不能把 "Hello" 转换成 int。
  2. 如果格式不正确，会抛出异常，程序会中止。
