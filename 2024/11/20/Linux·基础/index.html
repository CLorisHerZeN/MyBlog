Linux应用领域

![Linux 使用在那些地方](/img/Linux/Linux 使用在那些地方.jpg)

  • 服务器：Linux是服务器操作系统的首选，特别是在Web服务器、数据库服务器、文件服务器等领域。许多大型网站和在线服务都运行在Linux服务器上。
  • 嵌入式系统：Linux被广泛用于嵌入式系统，如智能手机、平板电脑、路由器、电视、汽车导航系统等。它的轻量级和可定制性使其成为嵌入式开发的理想选择。以后在物联网中应用会更加广泛。
  • 个人桌面：此领域是传统linux 应用薄弱的环节，近些年来随着ubuntu、fedora [fɪˈdɔ:rə] 等优秀桌面环境的兴起，linux 在个人桌面领域的占有率在逐渐的提高。

Linux介绍

  • linux 是一个开源、免费的操作系统，其稳定性、安全性、处理多并发已经得到业界的认可，目前很多企业级的项目(c/c++/php/python/java/go)都会部署到Linux/unix 系统上。

  • 常见的操作系统(windows、IOS、Android、MacOS, Linux, Unix)

  • Linux 主要的发行版：

    Ubuntu(乌班图)、RedHat(红帽)、CentOS、Debain[蝶变]、Fedora、SuSE、OpenSUSE

  • Linux 和Unix 的关系：

    Linux 和 Unix 是两种操作系统，Linux 是一种类 Unix 的操作系统，它们都是多用户、多任务的操作系统，Linux 还具有一些 Unix 所没有的特性，例如支持多种文件系统、支持多种硬件平台、支持多种编程语言等

    ![Linux 和Unix 关系图](/img/Linux/Linux 和Unix 关系图.jpg)

虚拟机操作

虚拟机克隆

如果已经安装了一台linux 操作系统，还想安装更多，没有必要重新安装，只需要克隆

  1. 方式1：直接拷贝一份安装好的虚拟机文件
  2. 方式2：右键虚拟机，使用vmware 的克隆操作，一般选择完整克隆 （注意， 克隆时需要先关闭linux 系统）

虚拟机快照

  • 在使用虚拟机系统时，想回到原先的某一个状态，如担心有些误操作造成系统异常，需要回到原先某个正常运行的状态，可以使用vmware 提供快照管理功能

  • 进入系统后右键虚拟机，选择快照-拍摄快照，记录状态

  • 进入快照管理器，点击想要转到的快照

虚拟机迁移和删除

  • 虚拟系统安装本质是文件(放在文件夹中)、
  • 迁移：把安装好的虚拟系统文件夹整体拷贝或者剪切到另外位置使用
  • 删除：用vmware 进行移除，点击菜单 --> 从磁盘删除即可，或者直接手动删除虚拟系统对应的文件夹

Linux目录结构

  • linux 的文件系统采用级层式的树状目录结构，在此结构中，最上层是根目录“/”，然后在此目录下创建其他的目录
  • 在Linux 里，一切皆文件(!!)

具体的目录结构(不用背)

  1. /（根目录）：所有目录和文件的最顶层目录
  2. /bin（二进制文件）：包含系统启动时必需的二进制文件，如ls、cp、mv等。是Binary 的缩写, 这个目录存放最经常使用的命令
  3. /sbin（系统二进制文件）：包含系统管理程序，如ifconfig、reboot等。s 是Super User 的意思，这里存放系统管理员使用的系统管理程序
  4. /home（用户主目录）：存放普通用户的主目录，在Linux 中每个用户都有一个自己的目录，一般该目录名是以用户的账号命名，如/home/user1、/home/user2等
  5. /root（根用户主目录）：根用户（超级用户）的主目录
  6. /lib： 系统开机所需要最基本的动态连接共享库，作用类似于Windows 里的DLL 文件。几乎所有的应用程序都需要用到这些共享库
  7. /lost+found：该目录一般情况下为空，当系统非法关机后，这里就存放了一些文件
  8. /etc（配置文件）：所有的系统管理所需要的配置文件和子目录, 如安装mysql数据库my.conf
  9. /usr（用户程序）：包含用户程序和文件，如/usr/bin（用户二进制文件）、/usr/lib（库文件）、/usr/share（共享文件）等。用户的很多应用程序和文件都放在这个目录下，类似与windows 下的program files 目录
  10. /boot（启动文件）：存放启动Linux 时使用的核心文件，包括一些连接文件以及镜像文件
  11. /proc（进程信息）：不能动，一个虚拟的目录，它是系统内存的映射，访问这个目录来获取系统信息
  12. /srv（服务数据）：包含特定服务的数据，如网站服务的数据
  13. /sys（系统信息）：包含系统信息，如设备驱动程序信息
  14. /tmp（临时文件）：包含临时文件，系统重启后这些文件会被清除
  15. /dev（设备文件）：包含设备文件，如/dev/sda（硬盘）、/dev/tty（终端）等
  16. /media（自动挂载点）：用于自动挂载的设备，如USB驱动器、CD-ROM等
  17. /opt（可选软件）：包含可选的第三方软件，如/opt/oracle、/opt/mysql等
  18. /usr/local：这是另一个给主机额外安装软件所安装的目录。一般是通过编译源码方式安装的程序
  19. /var（变量文件）：包含经常变化的文件，如日志文件、缓存文件、临时文件等
  20. /selinux [security-enhanced linux]：SELinux 是一种安全子系统,它能控制程序只能访问特定文件, 有三种工作模式，可以自行设置
